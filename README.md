# My Personal Website

## Stuff Used

- [Svelte](https://svelte.dev) and SvelteKit for static site generation
- [mdsvex](https://mdsvex.pngwn.io/) to convert Markdown content
- [PrismJS](https://prismjs.com/) for code syntax highlighting
- [RemixIcons](https://remixicon.com/) for icons (except `src/lib/assets/favicon.svg` which I made)
- [Fontsource](https://fontsource.org/) for fonts (Noto Sans, Noto Serif, Fira Code)

See `package-lock.json` for full list and details.
